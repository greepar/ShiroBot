## 项目结构

- `ShiroBot`: 主程序
- `ShiroBot.SDK`: 插件与适配器开发 SDK
- `ShiroBot.Model`: 共享模型
- `ShiroBot.DemoPlugin`: 标准示例插件
- `ShiroBot.DemoAdapter`: 标准示例适配器

## 构建

```powershell
dotnet build .\ShiroBot.slnx
```
